## 📖 About This Repository

This repo is home to all the Python scripts featured on **The Python Guy** YouTube channel.

The goal is simple — break down Python into short, practical hacks that anyone can follow, run, and learn from. Every script is self-contained, beginner-friendly, and built to work with minimal setup. No fluff, just code that does something cool.

New scripts drop every **Tuesday & Friday** alongside the video.

---

## 🚀 Quick Start

```bash
# 1. Clone the repository
git clone https://github.com/thepythonguy/how-to-series.git
cd how-to-series

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run any script
python script_name.py
```
